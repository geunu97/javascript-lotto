  - [x] 숫자 범위는 1~45이다.

  - [x] 6개의 번호가 있어야 한다.

## 📌 예외 발생 상황

- [ ] 1. 입력 받은 당첨 번호의 형식
  - [ ] 당첨 번호를 쉼표로 구분할 수 없는 경우
  - [ ] 당첨 번호의 갯수가 6개가 아닐 경우
  - [ ] 당첨 번호가 중복될 경우
  - [ ] 당첨 번호가 숫자가 아닐 경우

- [ ] 2. 입력 받은 보너스 번호의 형식
  - [ ] 보너스 번호가 1개가 아닐 경우
  - [ ] 보너스 번호가 숫자가 아닐 경우
